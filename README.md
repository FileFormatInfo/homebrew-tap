# FileFormatInfo Tap

## How do I install these formulae?

`brew install fileformatinfo/tap/<formula>`

Or `brew tap fileformatinfo/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "fileformatinfo/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
